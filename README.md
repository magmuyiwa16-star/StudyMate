# StudyMate
Interactive study and exam prep tracker for 400 MLS students in 2nd semester 
Simple, offline friendly html file
Use pomodoro timer to study in 45 minutes sessions
track exam deadlines
keep studying streaks, get recommendations on what to study daily based on the course contents and check off each topic in your courses to get updated exam readiness. 
how to use: 
download or open the 'StudyMate.html' file from the repository, open it in a browser, (optional) add it to your home screen for one tap access like an app.
